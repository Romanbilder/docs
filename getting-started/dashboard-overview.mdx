- **Profile Settings**: Update personal information

## Analysis History

### History Table Columns

| Column | Description | Actions |
|--------|-------------|---------|
| **Status** | Analysis completion status | View progress, retry failed |
| **Date** | When analysis was started | Sort by date, filter by period |
| **Source** | Code source (GitHub, paste, upload) | Re-analyze same source |
| **Type** | Analysis level (Mini, Standard, Ultra) | Compare different types |
| **Score** | Security score (0-100) | View detailed breakdown |
| **Issues** | Number of vulnerabilities found | Jump to specific issues |
| **Credits** | Credits consumed | Track spending |
| **Actions** | Available operations | Download, share, delete |

### Filtering and Search

<Tabs>
  <Tab title="Time Filters">
    - **Last 7 days**: Recent analyses
    - **Last 30 days**: Monthly view
    - **Last 90 days**: Quarterly overview
    - **Custom range**: Specific date periods
  </Tab>
  
  <Tab title="Status Filters">
    - **Completed**: Finished analyses
    - **In Progress**: Currently running

    - **Scheduled**: Future automated scans
  </Tab>
  
  <Tab title="Source Filters">
    - **GitHub**: Repository analyses
    - **Upload**: File-based analyses
    - **Paste**: Code snippet analyses
    - **API**: Programmatic submissions
  </Tab>
</Tabs>

### Bulk Operations

Select multiple analyses to:
- **Download reports**: Get multiple PDF/JSON reports
- **Delete analyses**: Clean up old results
- **Export data**: CSV export for external analysis
- **Share results**: Send to team members

## Dashboard Shortcuts

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + N` | New Analysis |
| `Ctrl/Cmd + H` | View History |
| `Ctrl/Cmd + S` | Search Analyses |
| `Ctrl/Cmd + R` | Refresh Dashboard |
| `Ctrl/Cmd + ,` | Open Settings |

### Quick Navigation

- **Breadcrumbs**: Navigate back through sections
- **Search**: Global search across all analyses
- **Favorites**: Bookmark frequently accessed analyses
- **Recent**: Quick access to last 5 analyses
